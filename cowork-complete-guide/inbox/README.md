# inbox/

**Зона входящих — сюда бросаешь всё новое.**

Это точка входа для всех необработанных файлов. Ничего здесь не хранится постоянно — всё должно быть обработано и перемещено в `processed/`.

## Как использовать
- Скопируй новый файл сюда
- Скажи Claude: *"Прочитай скилл file-processor и обработай мой inbox"*
- Готово — файлы будут разложены по местам
