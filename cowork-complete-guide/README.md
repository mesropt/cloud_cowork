# Система обработки файлов

Простая система для управления постоянным потоком файлов.

## Структура

| Папка | Назначение |
|-------|------------|
| `inbox/` | Входящие — сюда бросаешь всё новое |
| `processed/` | Обработанные файлы, разложенные по категориям |
| `outputs/` | Результаты работы Claude (отчёты, сводки) |
| `reference/` | Справочные материалы только для чтения |

## Как работает

1. Копируешь новый файл в `inbox/`
2. Говоришь Claude: *"Прочитай скилл file-processor и обработай мой inbox"*
3. Claude читает каждый файл, определяет тип, переименовывает и перекладывает в нужную подпапку `processed/`
4. Создаётся лог в `outputs/` — что и куда было перемещено
